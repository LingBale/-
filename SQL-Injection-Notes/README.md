# SQL 注入学习笔记

本目录包含 SQL 注入的完整学习资料，包括 PDF 笔记、本地测试环境以及 Python 脚本。

## 📂 目录结构

- `SQL-Injection-Notes.pdf`：SQL 注入详细笔记（40+ 页，含基础、进阶、WAF 绕过、工具用法等）
- `lab/`：本地测试环境文件（含漏洞页面、数据库脚本）
  - `SQL-Injection.php`：存在 SQL 注入漏洞的测试页面
  - `init.sql`：数据库初始化脚本（创建 test 库及 users 表）
  - `README.md`：环境搭建详细说明
- `code/`：Python 脚本集（布尔盲注、时间盲注、高级拓展脚本）
  - 基础脚本：`bool_*.py`、`time_*.py`
  - 高级拓展：`拓展_*.py`（多线程、断点续传、代理轮换、验证码识别）

## 🚀 快速开始

1. **搭建测试环境**：按照 `lab/README.md` 的步骤启动本地 Web 服务并初始化数据库。
2. **运行基础脚本**：进入 `code/` 目录，执行例如 `python bool_dbname.py`，观察输出。
3. **学习高级技巧**：参考 `code/` 中的拓展脚本，理解多线程、断点续传、代理轮换等实现。

## 📌 注意事项

- 所有脚本均基于 `http://localhost/SQL-Injection.php` 编写，判断依据为页面包含 `"User:"`（真）或 `"No user found."`（假）。
- 如需用于其他目标，请修改脚本顶部的 `TARGET_URL` 和判断逻辑。
- 时间盲注脚本中 `SLEEP_TIME` 必须大于 `TIMEOUT`，否则无法正确检测延时。
- 代理轮换脚本中的代理地址为示例，使用前请替换为真实可用代理。
- 验证码识别脚本需要额外安装 Tesseract OCR 引擎及 Python 库，详见脚本内注释。
